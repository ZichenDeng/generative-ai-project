## Current Project Direction

The project is now framed as a focused benchmark study rather than a full end-to-end antibody generation pipeline. The main question is whether pretrained antibody-aware or interaction-aware protein representations can improve prediction performance on biologically relevant antibody tasks such as binding or developability.

## Scope

This share repo intentionally excludes large raw data and internal draft history. It includes small processed fold files, benchmark manifests, and lightweight scripts needed for quick team alignment and milestone execution.
