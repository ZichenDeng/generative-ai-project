# Data

This directory collects the data-facing materials that are safe and useful to share in git.

## What Is Included

- `source-manifest-2026-04-09.md`: source list and role of each dataset or database
- `dataset-selection-2026-04-09.md`: current dataset decision memo
- `flab-task-manifest.csv`: recommended FLAb benchmark tasks and labels
- `processed/`: reproducible fold assignments for the first FLAb tasks
- `raw/flab/README.md`: instructions for FLAb raw task files
- `dataset-comparison-2026-04-10.md`: comparison of FLAb, GDPa1, and the Jain clinical-stage antibody dataset

## What Is Not Included

- large raw benchmark downloads
- gated third-party data files
- ad hoc local copies of external datasets

The intent is to keep this share repo lightweight and reproducible. Teammates can use the manifests and notes here to retrieve permitted source data separately.
